This app lets you manage Applicants, Contacts and Leads by their Phone Number in a few simple clicks

Select configuration and go to settings

In the Phone Calls section make sure to check on CRM Lead, Partner Call and Applicant Call

Then click on Save

In the Phone Calls Menu Click on New

A wizard opens that lets you create a new Applicant, Contact and Lead

The Caller Phone is a mandatory field

Fill in all the Details and then save it

A new Record with the Phone Number in international format is Created

Only the Users with given access at the group will be allowed to view the records

While filling the details you can also add or create Custom Tags for the calls

The tag becomes visible in the record

To view the records by tags go to configuration and select Call Tags

Records with associated tags are visible

To create Call Summary go to configuration and select Call Summaries

Now click on New to create a new Call Summary

A wizard opens. Enter the Call Summary Name and then save it.

A new Call Summary is visible

The Record created in Phone Calls is visible in Contacts

The Record created in Phone Calls is visible in CRM

The Record created in Phone Calls is visible in Recruitment
